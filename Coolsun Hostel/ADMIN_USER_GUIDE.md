# 📖 Hostel ERP - Operations Manual

## 1. New Tenant Entry (The Wizard)
**Note:** We do not use CSVs. Every tenant must be entered manually to ensure data quality.
1.  Go to **"New Admission."**
2.  **Select Room:** Check if it is Small, Medium, or Large. (Green = Vacant).
3.  **Upload Documents:** Take photos of their ID, the Agreement, and Police Form. You cannot hit "Save" without these!
4.  **Police Status:** If verification is not done, mark as **"Pending."** The system will remind you daily.

## 2. Managing Repairs (The Service Queue)
* **Urgent (Red):** Water leak, Power outage. Call Manager immediately.
* **Routine (Yellow):** Cleaning, Guard Ducts. Mark done daily.
* **Waitlist (Blue):** Broken handle, Loose hinge. Click **"Add to Carpenter Queue."**

## 3. Financial Rules
* **Owner's Personal Bills:** Select Category **"Owner Personal"** -> Note: "Home Bill". Upload Receipt.
* **Shared Rooms:** If Room 201 has 3 people, collect rent individually. The system tracks who paid and who is late.

## 4. Compliance Checks
* Check the **"Police Verification Widget"** every morning. If a tenant is red, ask them for the form today.
